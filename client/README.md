# Frontend

Aplicación Next.js con Tailwind CSS para consumir el backend de usuarios.

## Requisitos
- Node.js 18 o superior

## Instalación
1. Desde la carpeta `client`, instalar dependencias:
   ```bash
   npm install
   ```
2. Opcional: definir la variable `NEXT_PUBLIC_API_URL` en un archivo `.env.local` si el backend usa una URL distinta a `http://localhost:8080`.

## Ejecución
```bash
npm run dev
```

La aplicación quedará disponible en `http://localhost:3000`.
